## üìä USAGE INSTRUCTIONS

1. **Copy the relevant prompt** for the screen you need
2. **Paste into Google's Imagen/Gemini** (or Midjourney, DALL-E 3, etc.)
3. **Add this suffix** to all prompts for consistency:
   ```
   High-fidelity UI design, modern web design, clean interface,
   professional mockup, Dribbble quality, 4K resolution.
   ```
4. **Iterate** by adding specific requests:
   - "Show mobile version"
   - "Dark mode variant"
   - "Add more whitespace"
   - "Make it more premium/luxury"

5. **Design tokens to reference**:
   - Corner radius: 12px (cards), 8px (buttons), 24px (large containers)
   - Shadows: 0 4px 6px rgba(0,0,0,0.1) (subtle), 0 10px 40px rgba(0,0,0,0.15) (elevated)
   - Spacing: 8px base unit (multiples: 8, 16, 24, 32, 48, 64)
